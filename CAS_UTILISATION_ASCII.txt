## DÉTAIL DES CAS D'UTILISATION

### 1️⃣ S'AUTHENTIFIER
```
┌──────────────────────────────────────┐
│ S'AUTHENTIFIER                       │
├──────────────────────────────────────┤
│ ID: UC-001                           │
│ Acteur Principal: Utilisateur        │
│ Pré-condition: Compte valide         │
│ Post-condition: Token généré         │
├──────────────────────────────────────┤
│ FLUX PRINCIPAL:                      │
│  1. Saisir identifiant               │
│  2. Saisir mot de passe              │
│  3. Système vérifie credentials      │
│  4. Générer UUID token               │
│  5. Retourner token                  │
├──────────────────────────────────────┤
│ FLUX ALTERNATIF:                     │
│  • Credentials invalides:            │
│    └─ Tentative suivante (max 3)     │

│    └─ Fermer application             │
└──────────────────────────────────────┘
```
